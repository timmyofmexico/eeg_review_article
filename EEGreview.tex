\subsection {Вейвлет преобразование}
По аналогии с преобразованием Фурье вейвлет преобразование сигнала \(x(t)\) состоит в разложении сигнала по некоторому базису. Отличие заключается в том, что в качестве базисной выбирается "солитоноподобная",хорошо локализованная во временной и частотной областях функция \(\psi(t)\), обладающая рядом характерных признаков; базис формируется с помощью перехода к другому масштабу функции \(\psi(t)\) и её сдвигов вдоль временной оси. Непрерывное преобразование \(W\) функции \(f\) определяется формулой:

    \[W(a,b) = \frac{1}{\sqrt{a}}\int f(x)\psi\left(\frac{x -b}{a}\right)dx \]
    Использование локализованных функций позволяет проводить анализ процессов, характеристики которых меняются во времени, и обеспечивает двумерную развёртку сигнала \(x(t)\), при которой время и частота воспринимаются как независимые переменные \cite{29}. Поэтому данный метод хорошо подходит для локализации краткосрочных событий.  Detection of short pathological events as epileptic crises or normal ones as evoked potentials in EEG. Наиболее активно используется для  обнаружения P300 волн. Используется значительно реже, чем FFT, для online приложений, так как является более трудоемким в вычислительном плане. Он подходит для анализа нестационарных сигналов и представляет основные достижения спектрального анализа.
Пример использования для разпознавания эмоций представлен в
    \cite{12}, \cite{20}.
\subsection{Autoregressive model}

% EEG сигнал представляется, как авторегрессионная (AR).
Autoregressive (AR) model  -- модель временных рядов, в которой значение временного ряда в данный момент времени представляется, как линейная комбинация значений в предыдущие моменты времени.
    \[X(t) = a_0 + \sum_{i = 1}^p a_iX( t - i) +\epsilon_t, \]
    где \(\epsilon_t\) -- белый шум. Число \(p\) называется порядком модели.  AR модель примененная к сигналам продолжительности больше 2с, в силу нестационарности данного сигнала, может давать неадекватный результат.
     Однако, как сообщается в \cite{28} участок ЭЭГ сигнала продолжительностью в 2 сек. с большой долей вероятности можно считать стационарным. Поэтому сигнал разбивается  на элементарные непродолжительные сегменты -- и уже они рассматриваются, как AR модель.
     Другой способ учесть нестационарность ЭЭГ использовать адаптивную AR модель. Two of the commonly used adaptive algorithms are the
recursive-least-squares algorithm (RLS) and LMS , but the LMS algorithm was used here due to its computational simplicity. The difference of AAR to the AR model is that the parameters \(a_1, a_2, \ldots, a_p\) can vary with time \((a_i = a_i(t))\), however it is assumed that the parameters change only 'slowly' and this method doesn't suitable for strong nonstationarity \cite{41}.% на что лучше заменить strong

\section{Методы классификации}
После формирования качественного признакового пространства выбор классификатора, играет уже не столь значительную роль. Рассмотрим
\begin{itemize}
\item Байесовский классификатор --  класс алгоритмов классификации, основанный на принципе максимума апостериорной вероятности. Для классифицируемого объекта вычисляются функции правдоподобия каждого из классов, по ним вычисляются апостериорные вероятности классов. Объект относится к тому классу, для которого апостериорная вероятность максимальна. В \cite{3} в качестве функции правдоподобия для классифицированного объекта рассматривается смесь нормальных распределений. Пример байесовского метода для управления тележкой \cite{39}. Другим примером байесовских классификаторов являются классификаторы, основанные на HMM.  The HMM itself could be seen as a finite automata, containing \(s\) discrete states, emitting a feature vector at every time point that depends on the current state. Each feature  vector is  often modelled using \( m\) Gaussian mixtures per state. The transition probabilities between states are described using a transition matrix.  Для каждого класса обучается своя HMM, и затем при классификации оценивается вероятность условная вероятность сигнала, при условии модели соответствующего класса.  В \cite{42} предложена система правописания на основе двухклассового классификатора. На каждом этапе множество букв делится пополам. В зависимости от выбора пользователя остается только одна половина, которая делится снова делится пополам. В качестве распознаваемых сигналов выступали воображаемые движения и одна 
    mental task. Так же online распознавание воображаемых движений представлено в \cite{43}.
    Другим примером байесовского классификатора является QDF. Вероятность каждого класса, как и в LDA описывается нормальным распределением, но уже не требуется равенство матриц ковариации. Сравнение данного метода с HMM, NN, LDA и другими алгоритмами  на примере классификации ментальных  задач представлен в \cite{43}.
     
\item Линейный дискриминантный анализ (LDA или иногда FLD) -- специальный случай байесовской классификации. Классический LDA используется для разделения данных на два класса. Предполагается, что объекты обоих классов имеют нормальное распределение с одинаковыми ковариационными матрицами. На основе этого предположения разделяющая поверхность является гиперплоскостью. Данный метод является вычислительно простым и по этому очень популярен для online приложений. В \cite{14} регуляризованный LDA применяется для для синхронного Berlin Brain-Computer Interface (BBCI), которым может управлять человек с минимальной подготовкой. Данная система осуществляет управление при помощи воображения движений. Так же представлена Hex-o-Spell система ввода текста, базирующаяся на BBCI и позволяющая
    вводить символы со скоростью 6-8 символов в минуту.  %Примеры использования \cite{12}.
    Расширением стандартного FLD является Stepwise linear discriminant analysis.  SLDA performs feature space reduction
by selecting suitable features to be included in the discriminant function.
\item SVM (support vector machine) -- линейный метод классификации. Особыми свойствами метода опорных векторов являются: максимизация зазора между классами; разреженность -- положение разделяющей поверхности зависит от небольшой доли объектов обучения; сводится к задаче квадратичного программирования и эффективно вычисляется на большом числе объектов; легко обобщается на случай нелинейных разделяющих поверхностей путем введения ядер. SVM часто используется вместе с P300 потенциалами \cite{15}, \cite{17}, \cite{18}, \cite{10}. В \cite{15} осуществляется управление роботом-гуманоидом, а именно выполнение задачи состоящей их нескольких этапов: перемещение по заранее определенной области и поднятие %picking up.
    выбранного пользователем объекта и доставка его в одну из набора доступных позиций. Реализация SVM взята из библиотеки LibSVM \cite{16}.
\item Нейронные сети --  математическая модель,  построенная по принципу организации и функционирования биологических нейронных сетей — сетей нервных клеток живого организм. Большинство исследователей применяют нейронные сети для offline классификации сигналов.
    Стоит отметить, что нейронные сети склонны к переобучению. В зависимости от структуры различают различные типы нейронных сетей:
     \begin{itemize}
     \item  A feedforward neural network is an artificial neural network where connections between the units do not form a directed cycle. This class of networks consists of multiple layers of computational units, usually interconnected in a feed-forward way. Each neuron in one layer has directed connections to the neurons of the subsequent layer. In many applications the units of these networks apply a sigmoid function as an activation function. Пример использования \cite{8}.
         Сравнение MLP c одним скрытым слоем с  LDA  для задачи классификации представлен в \cite{41}. Отметим, что LDA показал результат  лучше, чем MLP.
     \item  Сверточная нейронная сеть -- в основе лежит идея, что  некоторые трансформированные версии одного и того же сигнала
     (изображения получающиеся друг из друга вращением) вызывали один и тот же выход. Данное свойство называется структурной инвариантностью (invariance by structure). В \cite{24} применяется для обнаружения P300.
     \item learning vector quantization (LVQ) --  neural network  applies a winner-take-all Hebbian learning-based approach. Distinction Sensitive Learning Vector Quantization (DSLVQ) улучшенная версия LVQ; в отличии LVQ,  DSLVQ оценивает при помощи весов компоненты входного вектора признаков в зависимости от их значимости для классификации \cite{26}.  В \cite{25} LVQ используется для  online классификации воображаемых движений, а в \cite{23} для синхронного online распознавания движений.
     %DSLVQ weighs the components of an input vector according to their significance of classification [
     \item radial basis function network is an artificial neural network that uses radial basis functions as activation functions чаще всего используется для обнаружения эпилептических припадков.
     \end{itemize}
\item Так же перечислим классификаторы используемые в BCI, но не получившие широкое распространение:
\item k-nearest neighbor (kNN) -- основная идея метода заключается в том, что объект присваивается тому классу, который является наиболее распространённым среди соседей данного элемента. Соседи берутся исходя из множества объектов, классы которых уже известны, и, исходя из ключевого для данного метода значения k вычисляется, какой класс наиболее многочислен среди них. В \cite{12}, \cite{20} при распознавании эмоций показал результат значительно лучший, чем LDA.
\item Использование статистических тестов
\end{itemize}
